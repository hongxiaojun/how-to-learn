# How to Learn

一个关于学习方法的概念解剖工具集合。

## Skills

### ljg-learn

深度概念解剖师，通过 8 个探索维度（历史、辩证、现象、语言、形式、存在、美感、元反思）解构任何概念，并将洞察压缩为顿悟。

**使用方法：**
- 直接说："帮我解剖一下 [概念]"
- 或者说："概念解剖 [概念]"
- 或者说："/ljg-learn [概念]"

**输出格式：**
- Markdown 格式
- Denote 风格文件名：`{timestamp}--概念解剖-{概念名}__concept.md`
- 保存到：`~/Documents/notes/`

## 示例

尝试解剖这些概念：
- 财富
- 刚需
- 思考
- 家教
- 伴侣
- 朋友
